Project Details

Smart Locks

The Smart Locks project is an advanced security solution built using Arduino Uno and a suite of sensors: RFID, keypad, 5V relay, lock solenoid, and a fingerprint sensor. This system enables secure, multi-factor authentication for unlocking doors. Users can access the lock via RFID cards, unique PIN codes, or biometric fingerprint verification, ensuring robust security. The project required custom circuit design, sensor calibration, and programming for seamless user experience and reliable operation, making it ideal for modern smart homes and offices.

Synergy Project

The Synergy Project is a cross-platform mobile application developed using Flutter (Dart). It allows users to unlock smart locks with a single tap on their smartphone. The app leverages WiFi connectivity to communicate securely with the lock hardware, providing remote access and real-time control. This project involved integrating IoT protocols, designing an intuitive user interface, and ensuring secure data transmission between the app and the lock system, making smart home access more convenient, secure, and future-ready.
